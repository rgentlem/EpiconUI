Abstract

This interface is designed to keep source literature, structured document representations, model-assisted querying, and reproducible R analysis in a single browser workspace.

Selecting chunk references can route context into the LLM panel and also expose linked data products for statistical work. The intent is to shorten the path from reading to analysis without losing provenance.
